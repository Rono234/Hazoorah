About Hazoorah

Game Overview

Hazoorah is an interactive puzzle game where players arrange images in the correct order using logic and clues. Drag and drop items onto the grid to organize them properly, then click Submit to check your solution.

Back Button: Click on the bear to go back to previous page.

Shuffle Button: Randomizes items for a fresh start.

Clear Button: Resets the entire grid so you can begin again.

Hint Button: The hint button provides the user feedback on their placement of the images. (1 hint for Easy, 2 for Medium, 3 for Hard).

Information Button: This button opens the Information Panel, where you can read the game overview, rules, feature explanations, and helpful instructions before playing.

Settings Menu: Adjust music and toggle the hint system.

Pause Button: Freezes the game and stops the timer.

Note-Taking Journal

A built-in journal where players can write down ideas, clues, or strategies on a virtual notepad.

New Game Button

Begins a brand-new game session, sending the player directly into Level 1.

Resume Button

Lets the user continue the exact level where they previously left off.
